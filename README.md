## How To Create A Piece
You Can Create a piece of chess board by select the color at the end of page
then select the box that you want to put it on 
and from keyboard press the first letter of piece

--------------

- k => King
- Q => Queen
- R => Rook
- B => Bishop
- N => Knight
- P => Pawn
